# Stock Price Movement Classifier ??

## Project Overview
This project predicts whether a stock price will go UP or DOWN using Machine Learning.

## Dataset
Historical stock price data containing Open, High, Low, Volume and Turnover.

## ML Models Used
- Logistic Regression (Final Model)
- Decision Tree (for comparison)

## Features
- Data preprocessing & EDA
- Feature scaling
- Model fine-tuning
- Streamlit web app deployment

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Joblib

## Deployment
The project is deployed on Streamlit Cloud and accessible via a web interface.

## Result
Logistic Regression achieved the best performance with ~85% accuracy.
